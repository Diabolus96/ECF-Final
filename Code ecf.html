Gamesoft est un studio de jeu vidéo français, développant des jeux vidéo. Ils proposent des jeux vidéo principalement de type RPG (jeu vidéo de rôle) : ce type de jeu permet au héros de pouvoir évoluer par l’intermédiaire de combat ou de résolution de quête. Ces jeux vidéo sont à destination principalement des ordinateurs, Si le jeu possède un score élevé (jeu étant très attendu par les joueurs), alors Gamesoft effectue une version console.
